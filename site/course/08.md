# 08 一页纸作业

前七课给了阅读顺序：质量价值 → 能力圈 → 生意 → 护城河 → 财报抓手 → 估值常识 → 风险。本课要求你把框架压成**一页纸**，以 Apple 为教学对象完成一次完整拆解练习。

> 教育材料，非投资建议。作业产出是阅读训练，不是投资备忘录里的下单段落。禁止写买入/卖出/加仓/目标价/仓位比例。

## 为什么是「一页纸」

信息过载时，长度伪装成深度。一页纸强迫取舍：

- 只能保留最关键的依赖关系；
- 必须露出「不懂」；
- 方便隔几个季度回看：当初假设哪条错了。

若你无法压缩，通常不是因为细节太重要，而是结构还没立住——回到第 3–5 课补课比继续堆粘贴更有效。一页纸写不通时，优先删「正确的废话」，留下可被下一季财报打脸的句子。

## 作业目标

在不查阅「荐股结论」的前提下，仅用公开信息与你的产品体感，完成一份 Apple「一页纸」：

1. 生意一句话 + 用户与付费关系；
2. 收入结构与你盯的依赖；
3. 护城河：主张、证据、反例各最短句；
4. 财务抓手：收入质量 / 毛利 / 经营利润 / FCF 各一句批注（标注报告期）；
5. 估值思考：市场可能在为什么付钱 + 哪条假设最脆（**无目标价**）；
6. 风险清单四格及以上；
7. 「我不懂的地方」至少三条。

数字凡引用，请注明来源类型与「教学简化 / 公开信息」属性。

## 推荐流程（约 60–90 分钟）

1. **15 分钟**：重读你在第 3–4 课写的生意与护城河草稿，删形容词。
2. **25 分钟**：打开最新年度或季度公开披露，只抓本课四个财务抓手，各写一句。
3. **15 分钟**：写估值思考——只写假设与失效条件。
4. **15 分钟**：填风险表与「不懂」。
5. **最后 10 分钟**：删到真正一页；朗读一遍，去掉任何「暗示操作」的句子。

## 评分标准（自评）

| 等级 | 特征 |
|---|---|
| 未合格 | 全是形容词；或出现买卖/目标价；或几乎无「不懂」 |
| 及格 | 有结构、有报告期、有至少两条可证伪风险 |
| 良好 | 护城河有反例；财务句能扣回生意依赖；估值段无定价结论 |
| 优秀 | 三个月后仍能用同一页纸做「假设复盘」，清晰看到哪条被打脸 |

## 提交与分享建议（可选）

若你与学习同伴互评，请互评「证据与可证伪性」，不要互评「看涨看跌」。同伴指出你某句无法打脸，或某句偷渡交易建议，都是有价值的反馈。

互评时可只问四个问题：生意依赖写清了吗？护城河有反例吗？财务句有报告期吗？估值段是否偷偷写成了点位？四个都过，这份一页纸就算完成训练目标。

## 一页纸示例结构（示意，请自行重写）

下面是**字段顺序示意**，不是标准答案，也不是对公司的评价：

1. **生意**：消费级硬件入口 + 服务与生态延伸（教学简化）。
2. **用户**：个人/家庭为主；开发者参与分发经济。
3. **收入**：硬件与服务分拆；盯换机与服务对装机的依赖。
4. **护城河**：转换成本主张 + 开发者生态提问；写清反例。
5. **财务**：四句抓手 + 报告期。
6. **估值思考**：久期与质量溢价假设；最脆的一条；无点位。
7. **风险**：竞争 / 监管 / 估值压缩 / 偏差各一行。
8. **不懂**：例如某地区趋势细节、某费用科目口径等。

若你的一页纸比示意更短、但证据更硬，反而更好。

## 课程收束

学完八课，你应能：

- 把质量价值与消息/短线区分开；
- 在能力圈内选择研究对象；
- 用用户、收入、单位经济读生意；
- 用网络效应 / 转换成本 / 规模经济提问护城河；
- 用收入质量、毛利、经营利润、FCF 抓财报；
- 理解倍数含义与失效，而不产出目标价；
- 用风险清单与行为偏差约束自我叙事；
- 用一页纸完成可复盘输出。

这不是终点，而是可重复的阅读节奏。换一家美股科技公司时，模板字段不用改，改的只是证据与「不懂」列表。完成后请打开工作表开始填写。
